	º Para compilar el proyecto existe un script "makefile", escribir "make" en la terminal.
	º Para ejecutarlo ./esqueleto_qt.
	º La interfaz permite visualizar puntos, aristas, caras y también caras con varios colores
	pulsando en las cajas correspondientes. En otro caso también se pueden visualizar de la
	siguiente manera:
		1) Puntos -> tecla <p>
		2) Aristas -> tecla <l>
		3) Caras -> tecla <f>
		4) Caras de varios colores -> tecla <c>
	º Para las figuras hay un selector de las mismas, además de teclas asignadas para las mismas:
		1) Tetraedro (figura extra) -> tecla <1>
		2) Cubo -> tecla <2>
		3) Pirámide -> tecla <3>
	º Para mover los ejes su utilizan las teclas up, down, right, left.
