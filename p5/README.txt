	º Para compilar el proyecto existe un script "makefile", escribir "make" en la terminal.
	º Para ejecutarlo ./practica_objetos_B4.
	º Se ha integrado en el proyecto la funcionalidad de leer textura.
	º Podemos visualizar de la siguiente manera:
	
		1) Puntos -> tecla <1>
		2) Aristas -> tecla <2>
		3) Caras -> tecla <3>
		4) Caras de varios colores -> tecla <4>
		5) Modo iluminación plana -> tecla <5>
		6) Modo iluminación suave -> tecla <6>
		
	º Para las figuras hay unas teclas asignadas para las mismas:
		1) Tetraedro (figura extra) -> tecla <T>
		2) Cubo -> tecla <C>
		3) Pirámide -> tecla <U>
		4) Objeto Ply -> tecla <B>
		5) Cono -> tecla <N>
		6) Cilindro -> tecla <L>
		7) Esfera -> tecla <E>
		8) Rotacion -> tecla <R>
		9) RotacionPly -> tecla <W>
		10) Helicoptero -> tecla <A>
		11) Animacion -> tecla <S>
		12) Textura -> tecla <T>
		13) Extrusión -> tecla <X>
	
	º Para la iluminación:
		1) Activar/Desactivar segunda luz -> tecla <I>
		2) Mover luz2 sobre el eje x -> tecla <O> y tecla <P>
	
	º Para la vista ortogonal:
		1) Activar/Desactivar vista -> tecla <F7>
	
	º Distintos movimientos del objeto articulado:
		1) Desde tecla <F1> .. hasta .. tecla <F6>
		
	º Para mover los ejes su utilizan las teclas up, down, right, left ó con el ratón manteniendo
	el click derecho.
	º Para hacer zoom +/- se utilizan las teclas repag y avpag ó girar la rueda del ratón.
	º Los materiales vienen definidos en la función "materiales" en objetos_B5.cc.
