	º Para compilar el proyecto existe un script "makefile", escribir "make" en la terminal.
	º Para ejecutarlo ./practica_objetos_B3.
	º Podemos visualizar de la siguiente manera:
		1) Puntos -> tecla <1>
		2) Aristas -> tecla <2>
		3) Caras -> tecla <3>
		4) Caras de varios colores -> tecla <4>
	º Para las figuras hay unas teclas asignadas para las mismas:
		1) Tetraedro (figura extra) -> tecla <T>
		2) Cubo -> tecla <C>
		3) Pirámide -> tecla <P>
		4) Objeto Ply -> tecla <O>
		5) Cono -> tecla <Q>
		6) Cilindro -> tecla <Y>
		7) Esfera -> tecla <S>
		8) Rotacion -> tecla <R>
		9) RotacionPly -> tecla <W>
		10) Helicoptero -> tecla <A>
		11) Animacion -> tecla <S>
	º Para mover los ejes su utilizan las teclas up, down, right, left.
	º Para hacer zoom +/- se utilizan las teclas repag y avpag.
